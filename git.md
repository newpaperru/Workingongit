 * ненумерованный список(маркер)

## Выделения 

Есть различные виды выделений, используемых в языки разметки Markdown. Приведу пример 5-х таких видов: 

1. *Курсив*
2. **Полужирный**
3. `Выделение bg-дом`
4. ~~Зачеркнутый текст~~
5. > Комментарий
